# Dataset Card

## Summary

arXiv Paper Insights is a public-safe release of:

- daily recommendation results for newly released arXiv papers
- sanitized structured insights extracted from papers

The goal is to support paper discovery, ranking, and scientific information extraction workflows.

## Sources

- LinxSci (`https://linxsci.com`) as the source platform
- arXiv identifiers and paper metadata
- recommendation pipeline outputs derived from arXiv papers on LinxSci
- snapshot features derived from public platforms such as GitHub, Hugging Face, and Semantic Scholar
- structured paper insights generated by the LinxSci processing pipeline

For any `arxiv_id`, open `https://linxsci.com/pdf/{arxiv_id}` to read the paper and view its insights on LinxSci.

## What Is Included

- recommendation records with scores and lightweight metadata
- sanitized paper insights without raw chunk provenance
- release-ready JSONL and optional Parquet exports

## Update Schedule

- the public dataset is updated every Monday
- each update covers the previous week's recommended arXiv papers
- insights are included when they are available in the LinxSci release pipeline

## What Is Excluded

- paper full text
- PDF files
- markdown exports of papers
- raw source chunks and provenance snippets
- figure and table image binaries

## Intended Uses

- ranking and recommendation
- scientific IE and summarization
- research assistant and agent pipelines
- temporal analysis of paper discovery signals

## Limitations

- some signals are snapshots and may become stale
- ranking scores reflect pipeline heuristics and weights
- structured insights can contain model or extraction errors
- institution extraction quality may be noisy for some papers
